# Useful meta commands for `psql` console

- Connect to database
  - `\c dbname`

- Describe available relations
  - `\d`

- Describe specified relation
  - `\d name`

- List console commands and options
  `\?`

- List available SQL syntax help topics
  - `\h`

- Specified SQL syntax help topic
  - `\h topic`

- Quit
  `\q`
